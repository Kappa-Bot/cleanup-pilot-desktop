## Migration Notes

- Keep the current data contracts stable while changing presentation components first.
- Move from page tabs to task steps where the user is making a decision.
- Use the same evidence and risk vocabulary everywhere.
- Treat Home and Smart Check as the primary product loop, not as a passive summary.
